## 🛡️ Anonymous Participation — How It Works

UHAKIX protects citizens who report corruption or submit evidence:

| Feature | How It Protects You |
|---------|-------------------|
| No Registration Required | Use platform without creating an account |
| Anonymous Tips | Report fake projects, missing funds, ghost workers |
| Encrypted Submissions | All data encrypted end-to-end |
| No IP Logging | We don't store your digital fingerprint |
| Tor/VPN Compatible | Access through any privacy tool |
| Blockchain Evidence | Your report is hashed on-chain but your identity is never stored |

### Why Anonymous Reporting Matters
When citizens can safely report:
- Fake road construction projects (money spent, no road built)
- Ghost workers on government payrolls
- Inflated procurement contracts
- Missing school infrastructure funds
- SHA/NHIF fraud

...without fear of retaliation → more evidence flows → more accountability → less corruption.

## 📋 How to Report a Fake Project

### Step 1: Identify the Issue
- A project that's "complete" on paper but nothing on the ground
- Funds allocated but no visible progress after 6+ months
- Over-invoiced items (KES 10M toilet, KES 50M fence)

### Step 2: Gather Evidence
- Take photos/videos of the site
- Note: Project name, location, contractor, date
- Optional: Share official tender documents

### Step 3: Submit Anonymously

**Via WhatsApp/Telegram:**
1. Open UHAKIX bot chat
2. Type: "Report fake project"
3. Upload photos → Add location pin → Submit

**Via USSD:**
1. Dial *XXX#
2. Select: "3. Report Corruption"
3. Select: "1. Fake Project"
4. Enter: Project code/location → Submit

**Via Web:**
1. Go to "Report" tab
2. Click "Anonymous Report"
3. Upload evidence → Submit → Get case number

### Step 4: Track Your Case
- You'll receive a unique case number (no personal info required)
- Check status anytime via case number
- UHAKIX AI cross-references your report with IFMIS, procurement, and AG data
- If verified → Report published publicly → Media alerted → EACC notified

## 🗳️ Election Day Verification

### For Every Kenyan:
1. **Before voting:** Check if your polling station is verified
2. **After voting:** Take photo of Form 34A (posted outside)
3. **Upload via:** WhatsApp, Telegram, Web, or USSD photo submission
4. **AI verifies:** Checks watermarks, signatures, station code, tally math
5. **Result added:** If verified → live national tally updates
6. **Consensus check:** Multiple reports from same station → higher confidence

### No Internet? No Problem
- USSD: Send station code + tally numbers via SMS
- Photo submission works on any WhatsApp-capable phone
- Zero cost to citizens
