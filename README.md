# 🇰🇪 UUHAKIX — Verify Everything
## Multi-Agent AI for Government Transparency & Electoral Integrity

> *"Kila shilingi ihesabiwe"* — Every single shilling must be counted.

UUHAKIX ("Uhakika" = Certainty + "Uuhakix" = Verify + "X" = AI) is Kenya's first AI-powered, multi-agent government transparency and electoral accountability platform. Built by CoHusdex multi-agent team.

We illuminate every corrupt deal, verify every vote, track every shilling — so citizens can hold power accountable without shedding blood.

---

## What UUHAKIX Does:

🔍 **Corruption Detection** — AI cross-references IFMIS, Treasury, PPRA, Company Registry, Parliament Hansard to find suspicious connections
📊 **Real-Time Oversight** — Citizens track government spending in real-time via web, USSD (*XXX#), WhatsApp, and Telegram
🗳️ **Election Integrity** — Vote counting verification via citizen photo uploads → AI verification → blockchain immutability → live public dashboard
📢 **Automated Reporting** — AI generates reports for media, civil society, and parliament automatically
🔗 **Blockchain Evidence** — All findings immutably stored on Polygon blockchain — tamper-proof, publicly verifiable

## Election Verification — How It Works

```
Citizen at polling station
   ↓
Takes photo of Form 34A (posted on wall outside)
   ↓
Uploads via USSD, WhatsApp, Telegram, or Web
   ↓
VERIFY AGENT + AI checks authenticity
 - Watermarks detected
 - Official signatures matched
 - Polling station code valid
   ↓
If VERIFIED → COUNT AGENT adds to national tally
If NOT VERIFIED → Flagged for manual review
   ↓
Blockchain stores verified result hash
   ↓
Public dashboard shows real-time verified count
```
